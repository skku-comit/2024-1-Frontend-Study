1. 터미널을 열어주세요!
2. cd b0xercat/4w
(입력한 디렉토리로 이동)
3. git checkout -b "b0xercat/1-study-tutorial"
(브랜치를 생성하고 해당 브랜치로 이동)
4. b0xercat 안에 5w 폴더 생성, 그 안에 index.html 파일 생성
5. index.html에 <p>I love CoMit</p> 입력
6. git add .
7. git commit -m "chore: add index.html"
8. git push
9. git push --set-upstream origin b0xercat/1-study-tutorial
(새로운 브랜치에서 첫 커밋을 할 때마다 뜰겁니다!)
